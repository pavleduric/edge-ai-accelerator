# 🧠 Edge AI Hardware Accelerator (Work In Progress)

## 📌 Project Overview
This project focuses on the hardware implementation of an energy-efficient Edge AI Accelerator designed for high-performance computing. The architecture is optimized at the Register-Transfer Level (RTL) to handle parallel matrix multiplications and neural network inference directly on hardware, maximizing throughput while minimizing power consumption.

## 🛠️ Tech Stack & Tools
* **Languages:** SystemVerilog
* **Tools:** Vivado, ModelSim/QuestaSim
* **Concepts:** Computer Architecture, Systolic Arrays, Hardware Acceleration, VLSI Design
